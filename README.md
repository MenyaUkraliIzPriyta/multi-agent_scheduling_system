# Multi-Agent Scheduling System

Система автоматической генерации университетского расписания. Проект позволяет создавать расписание занятий с учётом предпочтений преподавателей, доступности аудиторий и временных слотов.

## 📌 Возможности

- Загрузка данных о преподавателях, предметах и их нагрузке из JSON-файла
- Генерация расписания на семестр с учётом:
  - Доступности преподавателей (дни, временные слоты)
  - Фиксированных и предпочтительных аудиторий
  - Типов занятий (лекции, практики)
  - Максимального количества занятий в день
- Автоматическое распределение занятий по аудиториям и временным слотам
- Экспорт расписания в Excel (.xlsx) с форматированием:
  - Лекции — синий фон
  - Практики — зелёный фон
  - Свободные аудитории — обычный стиль


## ⚙️ Требования

- Java 17+
- Maven
- Зависимости:
  - Jackson (для работы с JSON)
  - Apache POI (для генерации Excel)
